# 配置管理系统重构 - 任务追踪

## 项目概述

将环境变量配置迁移到数据库存储，并提供前端设置页面管理。

**开始时间**：2026-04-01
**预计总工作量**：14-20 小时

## 任务列表

### 阶段 1：数据库层搭建
- [x] 1.1 更新 Prisma schema 添加 Setting 模型
- [x] 1.2 运行 Prisma 迁移创建数据库表
- [x] 1.3 扩展 db.ts 添加 setting 操作方法

### 阶段 2：工具函数层
- [x] 2.1 创建 encryption.ts 实现加密/解密功能
- [x] 2.2 创建 config.ts 配置访问层
- [x] 2.3 添加缓存机制和回退逻辑

### 阶段 3：后端 API 层
- [x] 3.1 创建 GET /api/settings - 获取所有配置
- [x] 3.2 创建 POST /api/settings - 批量更新配置
- [x] 3.3 创建 POST /api/settings/migrate - 迁移环境变量
- [x] 3.4 创建 GET /api/settings/export - 导出配置
- [x] 3.5 创建 POST /api/settings/import - 导入配置

### 阶段 4：前端页面层
- [x] 4.1 创建 settings 目录和主页面
- [x] 4.2 创建 SettingsForm 表单组件
- [x] 4.3 创建 ImportExportDialog 组件
- [x] 4.4 在 DevSidebar 添加设置入口

### 阶段 5：集成现有代码
- [x] 5.1 更新 vercel.ts 使用数据库配置
- [x] 5.2 更新 webhook.ts 使用数据库配置
- [x] 5.3 更新其他使用环境变量的代码

### 阶段 6：测试和部署
